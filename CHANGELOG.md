# Changelog

이 프로젝트는 [Semantic Versioning](https://semver.org/)을 따릅니다.

## [1.1.0] - 2026-04-05

### Added

- **CLAUDE.md 하네스 컨텍스트 등록** (Phase 5-4): 하네스 구성 후 프로젝트 CLAUDE.md에 에이전트 팀 구조·실행 규칙·진화 로그를 기록. 새 세션에서도 하네스가 즉시 인식되어 에이전트 팀 활성화 보장
- **후속 작업 지원** (Phase 5-5): 오케스트레이터 description에 후속 키워드 필수 포함, Phase 0 컨텍스트 확인 단계로 초기/부분재실행/새실행 자동 판별
- **하네스 진화 메커니즘** (Phase 7): 실행 후 피드백 수집 → 피드백 유형별 수정 대상 매핑 → CLAUDE.md 진화 로그 기록 → 자동 진화 트리거 (반복 피드백, 반복 실패, 사용자 우회 감지)
- 오케스트레이터 템플릿에 Phase 0 (컨텍스트 확인) 추가 — 에이전트 팀/서브 에이전트 모드 모두 적용
- 오케스트레이터 description 템플릿에 후속 작업 키워드 패턴 포함
- CLAUDE.md vs 오케스트레이터 역할 분담표 — 중복 방지 가이드

### Changed

- 핵심 원칙 2개 → 4개로 확장 (CLAUDE.md 등록, 진화 시스템 추가)
- 산출물 체크리스트에 CLAUDE.md 등록·Phase 0 컨텍스트 확인 항목 추가
- 오케스트레이터 템플릿 description에 후속 작업 트리거 키워드 기본 포함
